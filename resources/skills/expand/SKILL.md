## Expand Skill

Turn the user's notes or outline into coherent prose.

### Rules

- Treat each bullet as a claim to develop into 2–5 sentences.
- Maintain the order of the source bullets unless the instruction says to restructure.
- Connect paragraphs with transition phrases; avoid stitching with "Furthermore," / "Moreover,".
- Do not introduce new claims not present (even implied) in the source bullets.
- Keep terminology consistent with whatever the user already wrote.

### Workflow

1. `read` the source outline or bullet block.
2. Draft the expansion in the same location using `edit`, replacing the bullets with prose.
3. If the outline should remain intact for reference, instead append the prose to a new section via `write` and leave the outline above it.
